# 月鹿翻译

月鹿翻译是一个简单易用的 Visual Studio Code 翻译扩展，帮助开发者进行代码注释和文档的快速翻译。

## 功能特性

- 选中文本一键翻译为英语
- 支持代码文件中的注释翻译
- 简洁的用户界面，轻量且高效

## 使用方法

1. 在编辑器中选中需要翻译的文本内容
2. 右键点击选中内容，在上下文菜单中选择"翻译为英语"
3. 或者使用快捷键（待配置）触发翻译功能

## 系统要求

- Visual Studio Code 1.100.0 或更高版本
